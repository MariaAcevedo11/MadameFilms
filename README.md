# Madame Films

**Madame Films** is a web dashboard dedicated to celebrating films with female protagonist, the platform is around films led by women which deserve a dedicated space to be discovered, discussed, and celebrated.

Madame Films lets you explore a curated catalog of films, follow the careers of remarkable actresses, and engage with a community that shares your passion for female-driven storytelling.

### Features  

- **Browse & Discover** — Explore a curated catalog of movies with female protagonists, complete with detailed info, genres, and cast.
- **Like & Review** — Log in to share your thoughts through personal reviews.
- **Actress Profiles** — Dive deep into the careers and filmographies of the actresses behind the stories.
- **Stats & Insights** — Visualize the catalog through interactive charts: movies by genre, actress statistics, and more.
- **Community Reviews** — Stay up to date with the latest and most popular reviews from the community.
- **Admin Panel** — Administrators can grow the platform by adding new movies and actress profiles directly from the dashboard.

# Installation Guide

---

## Prerequisites

Before running Madame Films, make sure you have the following installed:

- **Node.js**: version `^20.19.0` or `>=22.12.0`
- **npm**: bundled with Node.js

You can verify your versions with:
```bash
node -v
npm -v
```

---

## How to Run the Project

### 1. Clone the repository
```bash
git clone https://github.com/MariaAcevedo11/MadameFilms.git
cd madamefilms
```

### 2. Install frontend dependencies
```bash
cd frontend
npm install
npm install chart.js swiper
```

### 3. Run the frontend in development mode
```bash
.../MadameFilms/frontend
npm run dev
```

### 4. Open the app

The dev server URL will be shown in the terminal — by default something like `http://localhost:5173`. Open it in your browser to start exploring MadameFilms.

---

You should now see the homepage !
